ÉTAPE 1: Ouvrez un terminal PowerShell/CMD
═════════════════════════════════════════

cd c:\xampp\htdocs\Gestion_de_Pharmacie


ÉTAPE 2: Exécutez la migration
═════════════════════════════════════════

php artisan migrate


ÉTAPE 3: Créez les utilisateurs de test
═════════════════════════════════════════

php artisan tinker

Copiez-collez ceci (une ligne à la fois):

User::create(['name' => 'Admin User', 'email' => 'admin@pharmacy.test', 'password' => bcrypt('password'), 'role' => 'admin', 'is_active' => true]);

User::create(['name' => 'Pharmacist User', 'email' => 'pharmacist@pharmacy.test', 'password' => bcrypt('password'), 'role' => 'pharmacist', 'is_active' => true]);

User::create(['name' => 'Vendor User', 'email' => 'vendor@pharmacy.test', 'password' => bcrypt('password'), 'role' => 'vendor', 'is_active' => true]);

User::create(['name' => 'Staff User', 'email' => 'staff@pharmacy.test', 'password' => bcrypt('password'), 'role' => 'staff', 'is_active' => true]);

exit


ÉTAPE 4: C'est Prêt! 🎉
═════════════════════════════════════════

Ouvrez votre navigateur et visitez:
http://localhost:8000
